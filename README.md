# 🕵️‍♂️ یاری سیخوڕەکە (Spy Master - Ultimate)

ئەمە پڕۆژەیەکی سەرچاوە کراوەیە (Open Source) بۆ یاری بەناوبانگی کۆمەڵایەتی "سیخوڕەکە" (Spyfall). ئەم یارییە بە تەواوی بە زمانی **کوردی** دروستکراوە و دیزاینێکی مۆدێرن و سەرنجڕاکێشی بۆ کراوە.

پڕۆژەکە تەنها لە یەک پەڕەی HTML پێکهاتووە (Single File)، واتە زۆر سووکە و بەبێ ئینتەرنێت کار دەکات.

## ✨ تایبەتمەندییەکان

*   **دیزاینی شوشەیی (Glassmorphism):** ڕووکارێکی مۆدێرن و جوان بە ئەنیمەیشنی نەرم.
*   **بە تەواوی ڕیسپۆنسیڤ:** گونجاوە بۆ هەموو جۆرە مۆبایل و تابلێتێک.
*   **سیستەمی دەنگ:** بەکارهێنانی (Web Audio API) بۆ دروستکردنی دەنگی کاریگەر بەبێ پێویستی بە فایلی دەنگی دەرەکی.
*   **بابەتە جیاوازەکان:** کۆمەڵێک بابەتی ئامادەکراو وەک (شوێن، کەرەستە، ئاژەڵ، پیشە، خواردن).
*   **وشەی تایبەت (Custom Words):** توانای زیادکردنی وشەی تایبەت لەلایەن بەکارهێنەرەوە.
*   **کاتژمێر (Timer):** بوونی کاتژمێرێکی پێشکەوتوو لەگەڵ ئەڵقەی ڕەنگاوڕەنگ.
*   **دیاریکردنی دەستپێکەر:** یاریەکە خۆی بە هەڕەمەکی دیاری دەکات کێ پرسیاری یەکەم دەکات.

> **تێبینی:** بۆ باشترین ئەزموون، واباشترە لەسەر مۆبایل یاری پێ بکرێت.

## 🎮 چۆنیەتی یاریکردن

1.  **ناوتۆمارکردن:** ناوی هەموو یاریزانەکان بنووسە.
2.  **ڕێکخستن:** ژمارەی سیخوڕەکان و کاتی یاریەکە دیاری بکە.
3.  **دابەشکردنی ڕۆڵ:** مۆبایلەکە دەستاودەست پێ دەکرێت، هەر کەسێک کلیک دەکات بۆ بینینی ڕۆڵەکەی (هاوڵاتی یان سیخوڕ) و دواتر دەیشارێتەوە.
4.  **دەستپێک:** دوای ئەوەی هەمووان ڕۆڵەکەیان بینی، کات دەست پێ دەکات.
5.  **ئامانج:**
    *   **هاوڵاتی:** دەبێت لە ڕێگەی پرسیار و وەڵامەوە سیخوڕەکە بدۆزێتەوە بەبێ ئەوەی وشە نهێنیەکە ئاشکرا بکات.
    *   **سیخوڕ:** دەبێت خۆی وەک هاوڵاتی دەربخات و هەوڵبدات وشە نهێنیەکە بزانێت لە قسەی ئەوانی ترەوە.

## 🛠 تەکنەلۆژیاکان

ئەم پڕۆژەیە بەم زمانانە دروستکراوە:

*   **HTML5:** بۆ پێکهاتەی پەڕەکە.
*   **CSS3:** بۆ دیزاین، ئەنیمەیشن و ڕووکاری شوشەیی.
*   **JavaScript (Vanilla):** بۆ هەموو فەرمانەکان و ژیربێژی یاریەکە (Logic).
*   **FontAwesome:** بۆ ئایکۆنەکان.
*   **Vazirmatn Font:** بۆ فۆنتی کوردی جوان.
