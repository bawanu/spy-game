## 🕵️‍♂️ یاری سیخوڕەکە - Spy Master 

ئەمە وەشانی نوێ و پێشکەوتووی یاری "سیخوڕەکە" (Spyfall)ـە. پڕۆژەکە بە تەواوی بە زمانی **کوردی** نووسراوەتەوە و دیزاینێکی مۆدێرنی "Ultimate"ـی بۆ کراوە بە بەکارهێنانی تایبەتمەندییە نوێیەکانی وێب.

پڕۆژەکە تەنها لە یەک پەڕەی HTML پێکهاتووە (Single File)، واتە زۆر سووکە، خێرایە و بەبێ ئینتەرنێتیش کار دەکات.

## ✨ تایبەتمەندییە نوێیەکان (New Features)

*   **دیزاینی پێشکەوتوو (UI/UX):** بەکارهێنانی دیزاینی شوشەیی (Glassmorphism) تێکەڵ بە ڕەنگی نیۆن و ئەنیمەیشنی تێکست (Shimmer effects).
*   **باکگراوندی زیندوو:** جوڵەی پارچەکان (Particles) لە پاشبنەما کە جوانییەکی تایبەت دەبەخشێت بە یاریەکە.
*   **سیستەمی دەنگدان (Voting System):** زیادکردنی قۆناغی دەنگدان لە کۆتایی یاریەکەدا. ئێستا دەتوانن لەسەر شاشەکە گومانلێکراو هەڵبژێرن.
*   **ژمارەی هەوڵەکان (Attempts):** دەتوانن دیاری بکەن چەند هەوڵتان هەیە بۆ دۆزینەوەی سیخوڕەکە (بۆ نموونە ١ هەوڵ یان ٢ هەوڵ).
*   **ڕاگەیاندنی براوە:** شاشەی تایبەت بۆ ڕاگەیاندنی بردنەوەی "هاوڵاتیان" یان "سیخوڕەکان" بە شێوازێکی سەرنجڕاکێش.
*   **دەنگی سینتێسایزەر (Advanced Audio):** بەکارهێنانی (Web Audio API) بۆ دروستکردنی دەنگی (کرتە، سەرکەوتن، زەنگ، کات) بەبێ پێویستی بە فایلی mp3.
*   **ڕیسپۆنسیڤ:** گونجاوە بۆ هەموو جۆرە مۆبایل و تابلێتێک (Android & iOS & Windows & Mac).

### ٣. بەکارهێنان وەک فایل:
تەنها فایلی `index.html` دابەزێنە و لە هەر بگەڕۆکێک (Chrome, Safari) بیکەوە.

> **تێبینی:** بۆ باشترین ئەزموون و کارکردنی دەنگەکان و لەرینەوە (Vibration)، واباشترە لەسەر مۆبایل یاری پێ بکرێت.

## 🎮 چۆنیەتی یاریکردن

1.  **ناوتۆمارکردن:** ناوی یاریزانەکان بنووسە (لانی کەم ٣ یاریزان).
2.  **ڕێکخستنەکان:**
    *   ژمارەی سیخوڕ.
    *   کاتی گفتوگۆ.
    *   ژمارەی هەوڵەکان (چەند جار بۆتان هەیە بە هەڵە کەسێک دیاری بکەن).
3.  **دابەشکردنی ڕۆڵ:** مۆبایلەکە دەستاودەست پێ دەکرێت، هەر کەسێک کلیک دەکات بۆ بینینی ڕۆڵەکەی (هاوڵاتی یان سیخوڕ) و دواتر دەیشارێتەوە.
4.  **یاری و گفتوگۆ:** دوای ئەوەی یاری دەستیپێکرد، پرسیار لە یەکتری بکەن.
5.  **دەنگدان (کۆتایی):**
    *   کاتێک کات تەواو بوو، شاشەی دەنگدان دەکرێتەوە.
    *   لەسەر ناوی ئەو کەسە کلیک بکە کە گومانت لێیەتی.
    *   ئەگەر سیخوڕەکە دیاری بکەن، هاوڵاتیان دەیبەنەوە.
    *   ئەگەر هەوڵەکانتان تەواو بێت و سیخوڕ نەدۆزرێتەوە، سیخوڕەکە دەیباتەوە.
