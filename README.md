# 🕵️‍♂️ یاری سیخوڕەکە - Spy Game (Online & Local)

ئەمە یاری "سیخوڕەکە" (Spyfall)ـە. پڕۆژەکە بە تەواوی بە زمانی **کوردی** نووسراوەتەوە و دیزاینێکی مۆدێرنی "Glassmorphism"ـی بۆ کراوە.

وەشانی نوێی یاریەکە ئێستا خاوەنی **Online Multiplayer**ـە، واتە دەتوانیت لەگەڵ هاوڕێکانت لە دوورەوە یاری بکەیت، یان وەک جاران بە یەک مۆبایل (Offline) یاری بکەن.

## ✨ تایبەتمەندییەکان (Features)

*   **🌐 یاریکردن بە ئۆنڵاین (New):** بەکارهێنانی تەکنەلۆژیای **PeerJS (WebRTC)** بۆ پەیوەستبوونی یاریزانان بەبێ سێرڤەری ناوەند. دروستکردنی ژوور (Room) و پەیوەستبوون لە ڕێگەی کۆدەوە.
*   **🎨 دیزاینی پێشکەوتوو (UI/UX):** بەکارهێنانی دیزاینی شوشەیی (Glassmorphism)، ئەنیمەیشنی تێکست، و باکگراوندی جوڵاو.
*   **📱 پشتگیری PWA:** خاوەنی `manifest.json`ـە، دەتوانیت وەک ئەپ دایبەزێنیت. (بەشی ئۆنڵاین پێویستی بە ئینتەرنێتە، بەڵام بەشی لۆکاڵ بەبێ ئینتەرنێت کار دەکات).
*   **🔊 دەنگی زیرەک & 📳 لەرینەوە:** بەکارهێنانی **Web Audio API** و **Vibration** بۆ ئەزموونێکی باشتر.
*   **🔄 گەڕانەوە بۆ یاری (Rejoin):** ئەگەر ئینتەرنێت پچڕا یان لە یاریەکە دەرچوویت، سیستەمەکە بە شێوەی ئۆتۆماتیکی دەتداتەوە بەستنەوە بە ژوورەکەوە.
*   **🗳 سیستەمی دەنگدانی پێشکەوتوو:**
    *   *لۆکاڵ:* دیاریکردنی گومانلێکراو بە دەست.
    *   *ئۆنڵاین:* هەر یاریزانێک لە مۆبایلی خۆیەوە دەنگ دەدات و ئەنجامەکان لای هەمووان دەردەکەوێت.
*   **⚙️ کۆنتڕۆڵی سێرڤەر (Host):** ئەو کەسەی ژوور دروست دەکات دەتوانێت یاریزان دەربکات (Kick)، ژوور دابخات (Lock)، یان کات ڕابگرێت.

## 🚀 شێوازەکانی یاریکردن (Game Modes)

### ١. یاریکردن بە یەک مۆبایل (Local Mode)
*   هەموو یاریزانەکان لە یەک شوێنن.
*   تەنها یەک مۆبایل بەکاردێت.
*   مۆبایلەکە دەستاودەست دەکرێت بۆ بینینی ڕۆڵەکان.

### ٢. یاریکردن بە ئۆنڵاین (Online Mode)
*   هەر یاریزانێک بە مۆبایلی خۆی یاری دەکات.
*   پێویستە هەمووان پەیوەست بن بە ئینتەرنێتەوە.
*   یەک کەس دەبێتە **Host** (ژوور دروست دەکات) و کۆدەکە دەدات بەوانی تر.

## 🎮 ڕێنمایی یاریکردن

### بۆ مۆدی ئۆنڵاین (Online):
1.  **دروستکردن:** "یاریکردن بە ئۆنڵاین" هەڵبژێرە، ناوی خۆت بنووسە و "دروستکردنی ژوور" داگرە.
2.  **بانگهێشت:** کۆدی ژوورەکە (Room Code) بدە بە هاوڕێکانت.
3.  **چوونەژوور:** هاوڕێکانت کۆدەکە دەنووسن و "چوونە ژوورەوە" دادەگرن.
4.  **دەستپێکردن:** کاتێک هەمووان هاتن، سێرڤەر (Host) دەتوانێت یاریەکە دەستپێبکات.
5.  **ڕۆڵەکان:** هەموو کەسێک لە مۆبایلی خۆیەوە ڕۆڵی خۆی دەبینێت (تەنها یەک جار کلیک دەکەیت).
6.  **دەنگدان:** لە کۆتاییدا، هەر یاریزانێک لە مۆبایلی خۆیەوە دەنگ دەدات کێ سیخوڕە.

### بۆ مۆدی لۆکاڵ (Local):
1.  "یاریکردن بە یەک مۆبایل" هەڵبژێرە.
2.  ناوی یاریزانەکان تۆمار بکە.
3.  مۆبایلەکە بدە دەست یاریزانی یەکەم، کلیک دەکات و ڕۆڵەکەی دەبینێت، دواتر کلیک دەکاتەوە بۆ شاردنەوەی و دەیدات بە کەسی دواتر.
